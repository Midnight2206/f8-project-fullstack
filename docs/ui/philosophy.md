# Design Philosophy

## Content-first
Content is the primary focus.
UI exists to support readability and interaction clarity.

## Minimal chrome
Avoid excessive borders, shadows, gradients, and decorations.
Use spacing and typography hierarchy instead.

## Functional motion
Animation must communicate state, feedback, hierarchy, or transition.
Never animate for decoration only.

## Dark-native
Dark mode is the default visual environment.
Light mode is secondary.

## Accessible by default
All interfaces must support keyboard navigation, readable contrast, and reduced motion.
