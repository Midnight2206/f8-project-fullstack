# Color Tokens

## Rules
- Never hardcode hex values in components
- Use CSS variables only
- Tailwind must map to tokens

## Background
- bg-base
- bg-elevated
- bg-overlay
- bg-subtle

## Border
- border-default
- border-strong
- border-subtle

## Text
- text-primary
- text-secondary
- text-tertiary
- text-inverse

## Semantic
- success
- warning
- error
- info

## Accent
- accent
- accent-hover
- accent-muted

## Dark Mode
Primary environment.

## Light Mode
Must preserve hierarchy and contrast.
